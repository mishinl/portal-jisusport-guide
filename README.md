# portal-jisusport-guide

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内收录了若干以静态网页形式呈现的内容，便于集中管理和访问。本仓库不针对任何特定域名或网站，仅为页面存档与展示之用。

## 目录说明

```
portal-jisusport-guide/
├── pages/              # 存放归档的 HTML 页面文件
├── assets/             # 页面所依赖的静态资源（图片、样式、脚本等）
├── README.md           # 本文件
└── LICENSE             # 开源许可文件
```

- `pages/` 目录下每个子文件夹代表一个独立的页面归档，文件夹名称即页面标识。
- `assets/` 目录存储页面通用的或各页面引用的资源文件，按需组织。
- 各页面均为独立的 HTML 文件，可直接在浏览器中打开，或通过静态服务器托管访问。

## 页面归档说明

- 所有页面均为纯前端实现，不依赖后端服务。
- 页面内容经过整理和归档，保持原始结构与样式，未做功能性修改。
- 每个页面均可独立使用，互不干扰。
- 归档版本可能包含多个历史版本，以文件名或子文件夹区分。
